# v0.7 / v0.8 / v0.9 三版本对比

> **PM 视角的纵向演进**：v0.7 让玩家 **爽**、v0.8 让玩家 **顺**、v0.9 让玩家 **连**。
> 每版都对应 1-2 个核心痛点。

| 维度 | v0.7 爽感与事件 | v0.8 操作与惊喜 | v0.9 羁绊与世界 |
|------|----------------|----------------|----------------|
| **核心命题** | 怎么让玩家一上来就爽？ | 怎么让移动端也顺？ | 怎么让主题不孤岛？ |
| **痛点** | 宠物太晚 / 缺一键 / 事件少 | 滑屏不便 / 蛋太丑 | 主题之间无互动 |
| **关键指标** | 首次出宠物时间 | 移动端使用率 | 12 主题通关率 |
| **新模块数** | 4 | 4 | 4 |
| **构建大小** | 115.41 KB gzip | 116.52 KB gzip | **118.34 KB gzip** |

---

## v0.7 爽感与事件（4 模块）

### 1. 宠物早产
- Lv.4 → 出蛋；Lv.5 → 孵化（2 分钟内必拿宠物）
- **PM 价值**：降低"还要玩 10 分钟才给宠物"的挫败感

### 2. 一键合并
- 贪心反复找最优方向滑动，最多 6 轮
- 一次按钮合并所有可合并棋子
- **PM 价值**：让"看棋友翻盘"这种爽感**主动可触发**

### 3. 8 种随机事件
- 宠物撒娇 / 元素共鸣 / 时间倒流 / 双倍时段 / 连击风暴 / 扭蛋狂欢 / 守护之盾 / 神秘访客
- 8% 概率 / 次合并触发
- 全屏卡片 + 倒计时进度条 + 右上角 buff 倒计时
- **PM 价值**：每次合并都可能有"惊喜"，避免单调

### 4. gacha_frenzy 免单
- 扭蛋狂欢事件期间下一次扭蛋免费
- **PM 价值**：让"运气"事件反哺扭蛋消耗

## v0.8 操作与惊喜（4 模块）

### 1. 方向键 Dpad
- 网格下方 4 个固定方向按钮 ↑↓←→
- 移动端单手可玩，保留滑动手势
- **PM 价值**：解决"在地铁上滑屏会切后台"问题

### 2. 消消乐 3 连锁动
- 3 个同色同级排成行/列 → 整条线粉红脉冲
- 一键合并按钮显示「3连×N」预提示
- **PM 价值**：把 2048 的"被动合并"做成消消乐的"主动发现"玩法

### 3. 4 款限定蛋壳
- 💎 钻石 / ⭐ 星辰 / 🌸 樱花 / 🌑 暗夜
- 5% 概率抽取，bonus 15-50 币
- 蛋壳渐变色 + 光晕动画
- **PM 价值**：让扭蛋有"收藏价值"，开蛋有"开盲盒"的爽感

### 4. 神赐事件
- 8% 概率直接给 Lv.6 棋子入网
- 玩家不再"永远抽 1 级"
- **PM 价值**：解决"为什么我合成 1 小时还是只有 1 级棋子"

## v0.9 羁绊与世界（4 模块）

### 1. 6 对主题羁绊
- 万物有灵 / 魔力回路 / 人间烟火 / 星际交响 / 城市蓝图 / 梦境潜游
- 双方区域都 Lv.10+ → 解锁被动 buff（+10% 收益 / +2% 暴击 / +5% 幸运）
- 面板里实时显示"哪一区还差几级"
- **PM 价值**：从"通关 1 区"到"通关 2 区有 buff"的主线感

### 2. 世界进度环
- 12 主题按段绘 SVG 环
- 凑齐整圈（12 区都 Lv.11）→ 解锁「传说蛋」彩蛋
- **PM 价值**：长线目标 + 整圈奖励的"集齐"快感

### 3. 视觉打磨工具
- EASE 缓动统一（pop/slide/bounce/smooth/elastic）
- `burstParticles()` 通用粒子喷射
- `floatText()` 通用浮动文字
- **PM 价值**：未来加新功能可直接复用，无需重新打磨

### 4. 主题羁绊入口
- 更多 Tab 顶部加 🔗 主题羁绊入口
- 实时激活数显示
- **PM 价值**：让"两个区域都玩"成为可见的、有奖励的目标

## 关键技术决策

| 决策 | 理由 |
|------|------|
| 宠物从 Lv.10 → Lv.4 | 行为学上 2 分钟内必给正向反馈 |
| 8% 事件触发率 | 约 1/12 滑动触发一次 → 频繁但不骚扰 |
| 5% 限定蛋率 | 比"5 星武器"略低，更稀有感 |
| 8% 神赐率 | "刚刚差点放弃突然来一个"的逆转感 |
| Dpad + 滑动手势并存 | 桌面/移动端都能用 |
| 主题羁绊 buff 数值保守（10%/2%）| 避免单一 buff 改变游戏节奏 |
