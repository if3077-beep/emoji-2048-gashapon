# v0.4 / v0.5 / v0.6 三版本对比

> **PM 视角的纵向演进**：v0.3 优化爽感 → v0.4 拉留存 → v0.5 加深度 → v0.6 做社交。
> 每版都建立在前一版的"变现"或"留存"骨架上。

| 维度 | v0.4 留存与日活 | v0.5 成就与深度 | v0.6 社交与分享 |
|------|----------------|----------------|----------------|
| **核心命题** | 怎么让玩家明天还来？ | 怎么让玩家玩 1 小时不腻？ | 怎么让玩家主动拉新人？ |
| **关键指标** | DAU / 7 日留存 | 时长 / 成就完成率 | 分享率 / 排行榜上榜率 |
| **新模块数** | 4 | 4 | 5（含 1 项 CI 修复） |
| **构建大小** | 104.97 KB gzip | 109.78 KB gzip | 113.02 KB gzip |
| **依赖增量** | — | — | +@types/node |

## v0.4 留存与日活（4 模块）

### 1. 7 日签到
- 奖励表按指数递增：10/30/50+1扭/80/100+1神/150/200+太阳冠
- 主页 🎁 按钮带"待签"红点（小红球+动画脉冲）
- 连续中断 → 重新开始
- **PM 价值**：每天都有 1 个 1 秒钟的"我回来了"决策点

### 2. 回归奖励
- 24h+ 未见 → 欢迎弹窗
- 奖励按天数分级：1-2 天小礼 / 3-6 选中礼 / 7+ 大礼
- **PM 价值**：避免"消失 3 天就再也不回来"的死亡螺旋

### 3. 7 日挑战
- 10 种目标（合成 20/达 Lv.8/解锁 2 区/连击 ×10/扭蛋 10/喂食 3/抚摸 5/觉醒 1/集齐 1 区 11 等级/完成 3 任务）
- 进度条实时反馈
- **PM 价值**：让玩家"今天有 3 件事可做"，降低空白焦虑

### 4. 数据面板
- 累计扭蛋 / 总合成 / 总消耗 / 总收益 / 最佳连击
- 7 日活跃度 SVG 折线图
- **PM 价值**：让玩家看到"自己成长了"，强化继续玩的动力

## v0.5 成就与深度（4 模块）

### 1. 40+ 成就系统
- 6 大类：merge(8) / collect(8) / gacha(5) / pet(8) / combo(5) / awaken(5)
- 稀有 → 史诗 → 神话 → 传说 四档品质
- 进度条 + 解锁提示 toast
- **PM 价值**：覆盖所有玩家行为路径（爱合成的 / 爱收集的 / 爱抽卡的 / 爱宠物的）

### 2. 7 套宠物装扮
- 👑 王冠（基础）/ ☀️ 太阳冠（限定 - 通关全部 12 区）/ 🌸 樱花 / 🎉 派对 / 🧙 法师 / 🌵 仙人掌 / 🌈 彩虹
- 装扮 overlay 在 Pet 组件上显示
- **PM 价值**：让宠物成为玩家的"身份标识"（朋友圈炫耀点）

### 3. 3 首 BGM
- 日常 casual（轻快）/ 战斗 battle（紧张）/ 觉醒 awaken（史诗）
- Web Audio API 合成（无音频文件）
- **PM 价值**：背景音让 30 分钟的"心流"成为可能

### 4. 设置面板
- 音乐 / 音效 / 震动 / 重置
- 持久化偏好到 localStorage
- **PM 价值**：尊重用户习惯（地铁/办公室场景静音需求）

## v0.6 社交与分享（5 模块）

### 1. 分享卡片生成
- Canvas 540×960 PNG（9:16 竖版，朋友圈/小红书/抖音 友好）
- 内容：emoji + 等级 + 主题 + 宠物 + 合成次数 + 最佳连击 + 主题水印
- 按钮：📥 下载 PNG / 📋 复制分享文案
- 入口 2 处：更多 Tab + 图鉴 Lv.10+ 节点
- **PM 价值**：高质量图片 → 朋友圈/小红书传播 → 自然拉新

### 2. 本地排行榜
- 3 个榜单：🔥 最高连击 / ⭐ 最高等级 / 🏆 通关主题
- localStorage 存多玩家
- 序号彩牌（🥇 金 / 🥈 银 / 🥉 铜）
- **PM 价值**：单机游戏也能"较劲"，延长游戏寿命

### 3. 周末双倍活动
- 周六/周日全事件 coin ×2
- 主页 buff 条加 "🎉 周末双倍" 动画标识
- **PM 价值**：把"什么时候玩"问题变成"周末玩更划算"，激活低频用户

### 4. 图鉴展示墙
- 12 主题选择 + 树视图
- 点击已解锁 emoji → `NodePreview` 弹层（大图 + 故事）
- Lv.10+ 节点解锁"🖼️ 分享这张"按钮
- **PM 价值**：把图鉴从"工具"变成"成就展"，配合分享形成闭环

### 5. CI 修复
- 补 `@types/node`（vite.config.ts 的 `node:path`/`__dirname`）
- **PM 价值**：保证未来加更多 Node-side 工具不会重复踩坑

## 关键技术决策

| 决策 | 理由 |
|------|------|
| localStorage 存排行榜 | 单机游戏不上后端，降低复杂度；用户仍可手动"上榜" |
| Canvas 而非 DOM 转 PNG | 跨平台兼容性好，iOS Safari 也支持 |
| 周末双倍写在 `weekend.ts` | 一行 `applyWeekendBonus(coins)`，侵入性最小 |
| 装扮只改 emoji overlay | 不破坏宠物动效，复用现有渲染 |
| BGM 用 Web Audio 合成 | 0 资源包，秒开，无版权问题 |
